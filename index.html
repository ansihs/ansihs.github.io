About this game

Arrow hero is a minimalist game where your goal is to match a continuous, unstoppable stream of arrows using your keyboard keys.

This means that if you have more than one consecutive you only have to press once to validate both.

You can pause the game by pressing Space.
This game is mobile friendly.
